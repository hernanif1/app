# Hello World App

Minimal React app with Playwright e2e tests.

## Testing PR Comments

This branch is for testing PR comment functionality.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Run E2E Tests

```bash
npm run test:e2e
```

## GitHub Actions

This repository uses a reusable GitHub Action from `hernanif1/action` to:
- Run e2e tests
- Generate test reports
- Publish reports to GitHub Pages
- Add report links to PRs
